Looping in Ruby


First, lets start with 'What is a loop?' Computer programming languages, such as ruby, often need to execute the same block of code multiple times. Instead of repeating yourself over and over again, you can use a loop. Now lets learn about some of those loop statements!


While

A while loop will continue to complete the code while the condition remains true. When the condition becomes false, the loop will stop.

Example


Until

A until loop will continue to complete the code until the condition is met (true). After the condition is met, the loop will stop.

Example


Times

A times loop will run the code however many times is input by the user.

Example


Each

The each loop will take a list of variables and run the code for each of them. Once all of the variables have been run through the code, the loop will stop.

Example


For

The for loop, otherwise known as the for/in loop, will complete the code for all variables in the stated collection. When there are no more specified variables, the loop will stop.
