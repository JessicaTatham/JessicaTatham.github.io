Class Is In Session

Being that Ruby is an object-oriented language, everything you use in Ruby is an object. You use the object to do whatever it is that needs to be done, but in order for that to happen, the object needs to hold vital information that allows it to accomplish the tasks. This is where classes come in.

Rubylearning.com states it best: "Think about the things the object knows and the things the object does . Things an object knows about itself are called instance variables. The things an object can do are called methods."

Take a look at my example bellow. Our Class, Animals, is the object, it contains variables and methods that defines it's purpose in life. The instance variable, animal_type, is the data that the object is provided with. The object can take that data and put it through some methods to receive a desired output.
