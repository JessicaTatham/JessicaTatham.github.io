#map

First things first, the map method is a loop. I discussed a couple of loops in last weeks blog that all do very similar things, and the map loop is no different. It's main function is looping through the input and performing some function on each element of the input. #map is an enumerable. Don't know what that is? I didn't either! So of course, I googled it :) From what I can understand from the very unhelpful definition in the ruby docs, an enumerable is a class of methods that can be assigned any element as the input. Probably that isn't exactly right, but for now, it's the best I've got! So as an Enumerable#map, the method transforms each element in the input however you want. For example
